### 3.2 패턴별 미태깅 유형

#### 유형 1: \`[종목명/투자의견]\` 프리픽스
```text
[세아베스틸/Buy] 판매량 회복이 관건
[기아/Buy] 3Q24 Review: 밸류는 싼데, 실적은 좋아요
[현대차/Buy] 3Q24 Review: 충당금 한 스푼, 성과급 한 스푼
```
→ `RE_BRACKET_STOCK`이 종목명을 추출해도, 본문에 TAG_PATTERNS 매칭이 안 됨.

#### 유형 2: 감성적/시적 제목
```text
쨍하고 돌아오는 해뜰 날
겨울잠 다 잤습니다
다른 것을 다르게 볼 수 있어야 한다
작용과 반작용
```
→ 정규식으로는 태그 추출 불가. LLM fallback 필요.

#### 유형 3: \`(Not Rated)\` 포함
```text
에이에스텍 (453860/Not Rated)수출 회복을 기다리며
필에너지 (378340/Not Rated)보릿고개 끝자락
```
→ "Not Rated" 키워드가 TAG_PATTERNS에 없음.

#### 유형 4: 산업 prefix \`[/]\` 슬래시 포함
```text
[철강/금속] 알루미늄 공급 수난 시대
[통신] SK텔레콤, NAVER. 너 내 깐부가 돼라
```
→ `RE_BRACKET_SECTOR`가 `/[가-힣A-Za-z]+/` 로만 되어 `/`를 못 잡음.

#### 유형 5: 해외 종목 티커
```text
중지쉬창(300308.CH): 차세대 광트랜시버
넥스트파워 (NXT.US): 태양광 발전 Total Solution
```
→ 중국/미국 티커 포맷 미지원.

#### 유형 6: 분기/연도 마커 포함
```text
3Q24 Review: ...
2Q26 PREVIEW: ...
FY28부터 결실
```
→ 분기 실적 패턴이 직관적이지 않음 (현재 `Preview|preview`, `Review|review`만 매칭).

## 4. 개선 계획

### 4.1 Quick Wins (난이도 하, 즉시 적용)

| # | 항목 | 내용 | 예상 효과 |
